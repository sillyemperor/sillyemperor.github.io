Python3语法速查

Published: Fri 06 September 2019
By Wang Jiang

In Python.

tags: Python

一. 变量

变量就像一个盒子，你可以把具体的内容装进去。变量需要一个名字，例如：x，然后需要给她装点东西，装东西的方法很简单，用=符号，如果我们想把1装进x里面，像这样：x=1。

  1. 我想创建整数变量
x = 1 # 现在x里面装的是整数1
y = 2 # 现在y里面装的是整数2
  1. 我想创建字符串变量
h = '你好' # 现在h里面装的是字符串'你好'
name = '王小明' # 现在name里面装的是字符串'王小明'
  1. 我想创建一个新变量，用另外一个变量初始化他
x = 13
y = x

二. 类型

python支持自己定义类型，不过python也有预先定义好的类型，最基础的是：整数（int），小数或者浮点数（float），字符串（str），布尔（bool），空（NoneType），数组（list），字典（dict）。这些都是经常会用到的类型。

那么类型到底是什么呢，为什么要区分类型呢。首先是存放的方式不同，python对不同的内容需要采用不同的保存方式，就像水用桶装装盖个盖子就好了，酒需要密封，冰就需要隔热。所以python首先需要区分出类型然后根据类型设计存放的方式。然后是使用的方式不同，水可以用来洗碗洗衣粉，煮饭。酒一般只是用来饮用。冰用来冷敷伤口。整数和小数可以进行加减乘除运算，字符串就不行。

这里重点说一下布尔型，不同于整数，字符串有无数的值，布尔型只要两个值，真（True）和假（False）。当我们需要进行判断的时候我们就需要布尔型了。

  1. 我怎么知道一个变量是什么类型，可以使用函数type。
x = 1
print(type(x)) # 输出<class 'int'>，表示是整数
x = 1.0
print(type(x)) # 输出<class 'float'>，表示是小数
x = 'hello'
print(type(x)) # 输出<class 'str'>，表示是浮点数
x = True
print(type(x)) # 输出<class 'bool'>，表示是布尔型
x = None
print(type(x)) # 输出<class 'NoneType'>，表示是空
x = [1,2,3,4]
print(type(x)) # 输出<class 'list'>，表示是数组
x = {
    "name":"小明",
    "age":"13",
}
print(type(x)) # 输出<class 'dict'>，表示是字典
  1. 怎么把小数转变成整数，使用函数int。
x = 1.0
print(type(x)) # 输出<class 'float'>
y = int(x)
print(type(y)) # 输出<class 'int'>
  1. 怎么把字符串变成整数，用函数int。
x = '2019'
print(type(x)) # 输出<class 'str'>
y = int(x)
print(type(y)) # 输出<class 'int'>
  1. 怎么把其他类型变成字符串，用函数str。
x = 1
y = str(1)
print(type(y)) # 输出<class 'str'>
  1. 怎么把其他类型变成布尔型，用函数bool。
x = 1
y = bool(1)
print(type(y)) # 输出<class 'bool'>
  1. 整数到布尔型的转换，只有0是假的，其他都是真的
print(bool(1)) # True
print(bool(0)) # False
print(bool(-1)) # True
  1. 浮点数到布尔型的转换，只有0是假的，其他都是真的
print(bool(1.0)) # True
print(bool(0.0)) # False
print(bool(-1.0)) # True
  1. 字符串变成布尔型，只有空字符串是假的，其他都是真的，包括空格。
print(bool('Hello')) # True
print(bool('')) # False
print(bool('   ')) # True
  1. 数组变成布尔型，只有空数组是假的，其他都是真的。
print(bool([1,2,3])) # True
print(bool([])) # False
print(bool([False])) # True
  1. 字典转换成布尔型，空的字典是假的，其他都是真的。
print(bool({'name':'小明'})) # True
print(bool({})) # False
  1. 空类型无论如何都是假的
print(bool(None)) # False

三. 判断语句

小明和小红谁的个子高，汽车和火车谁跑得快，我的数学期末考试成绩有没有超过95分。当你思考这些事情的时候，你已经在执行判断语句了。

  1. 如何判断两个数的大小，用>,<符号
x = 1
y = 2
if x > y:
  print('大于')
# 结果是什么都不执行
if x < 2:
  print('小于')
#
  1. 如何判断两个数是否相等，用==判断是否相等，用!=判断是否不想等
x = 1
y = 1
if x == y:
  print('相等')
if x != y:
  print('不相等')
if 'hello' == 'Hello':
  print('如果你看到这段话，说明你遇到了假的Python')
  1. 如何判断一个班里是否有某位同学，包含关系用in判断。
x = 12
y = [1,2,3,4,5,12]
if x in y:
  print('包含')

x = '小明'
y = ['小红','小刚','小明','小强']
if x in y:
  print('包含')

if 'A' in 'abcdefg':
  print('如果你看到这段话，说明你遇到了假的Python')
  1. 判断语句的pythonic写法
x = 12
y = 7
z = None
if y > x:
  z = y
else:
  z = x

上面的语句最后的执行结果是z等于x,y中间大的那个，整个判断一共5行，而且不得不初始化一次z。
下面是完全等价的代码，但是代码明显减少而且更清晰。

x = 12
y = 7
z = y if y > x else x
